# glad1olus86 plugins

Claude Code plugin marketplace by [Heorhii Priadkin](https://priadk.in).

## Installation

```
/plugin marketplace add glad1olus86/plugins
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [mismatch-auditor](./mismatch-auditor/) | Blocks bash commands that mismatch the agent's stated intent via LLM verification |
